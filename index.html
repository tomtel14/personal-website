About

My working background lies in engineering, specifically in the automotive industry where I’ve worked for manufacturers including Ford and Rolls-Royce.

When my last contract came to an end in December 2019, I took some time off work to go travelling before planning my next move. When I returned, I started helping my parents run their online e-commerce store using a content management system and basic HTML. This exposure to coding fascinated me and made me realise I was ready for an exciting new challenge.

I made the decision to study front-end web developing because I’ve always been very methodical and have a high attention to detail. The idea of creating visually appealing and intuitive apps/web pages is definitely something which excites me and I’m really enjoying the learning process!

My time as an engineer gave me plenty of experience in working as a team where clear communication and problem-solving skills were essential. I’m now eager and ready to transition into web-developing where I can apply these skills as well as continue to learn and experiment with the latest tech every day!

Tech

Some of the tech I've enjoyed using so far:

  • HTML

    I have a good understanding and appreciation of the benefits of writing semantic HTML so I always do my best to write clean and easy to read HTML.

  • CSS

    Getting to grips with the layout techniques which CSS has to offer has been great! Using Flexbox and Grid in combination with media queries has enabled me to create responsive web apps.

  • SASS

    Learning SASS massively helped me organise and tidy up my CSS code, making it far more manageable and easier to work with.

  • BEM

    BEM works really well together with SASS and has further improved the maintainability of my code. I really like how it eliminates specificity issues.

  • Bootstrap

    My first CSS framework I experimented with. The grid system is excellent and makes it easy to quickly knock up responsive webpages which are easy on the eye.

  • JavaScript

    I’ve really enjoyed learning the many concepts behind JavaScript. The more I use it to interact with the DOM, the more I can see it's potential.

  • React JS

    Definitely the technology I’m most excited about. I’ve started creating components with React to build simple user interfaces and have begun working with state within components.

  • DevTools

    Basically the swiss army knife of front-end development. Perfect for debugging and performance optimisation.

  • VS Code

    My favourite editor. Easy to use without being overly complicated. The integrated terminal is handy too!
